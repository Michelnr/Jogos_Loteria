# Gerador de Jogos - Loteria Federal

Este projeto é um utilitário desenvolvido em Python para gerar combinações de números aleatórios para os principais jogos da Loteria Federal Brasileira. O objetivo principal é servir como ferramenta de estudo e automação para apostadores.

## 🚀 Funcionalidades

O sistema permite a geração de apostas para as seguintes modalidades:

*   **Mega Sena**: Gera 6 números únicos entre 1 e 60.
*   **Loto Fácil**: Gera 15 números únicos entre 1 e 25.
*   **Quina**: Gera 5 números únicos entre 1 e 80.

## 📋 Pré-requisitos

Para executar este projeto, você precisará apenas do:
*   [Python 3.x](https://www.python.org/) instalado.

## 🔧 Instalação e Execução

1. Clone o repositório ou baixe os arquivos do projeto.
2. Navegue até a pasta do projeto via terminal/prompt de comando.
3. Execute o script principal:

```bash
python Loteria_federal.py
```

## 🕹️ Como Usar

1. Ao iniciar, o programa exibirá um menu com as opções de jogos.
2. Digite o número correspondente ao jogo desejado (1, 2 ou 3).
3. Informe a quantidade de jogos (apostas) que deseja gerar.
4. O sistema imprimirá na tela as combinações geradas, todas organizadas em ordem crescente para facilitar o preenchimento dos volantes.
5. Para encerrar o programa, escolha a opção **4 - Sair**.

## 🛠️ Tecnologias Utilizadas

*   **Python**: Linguagem de programação.
*   **Random**: Biblioteca nativa para geração de números pseudo-aleatórios.
